About Novel Dialogue Enhancer

Novel Dialogue Enhancer is a browser extension that uses AI to improve translated novel dialogues, making them sound more natural and fluent.

Version:
Requirements: Ollama must be installed and running on your local machine.

How to Use:

  1. Install and set up Ollama on your computer
  2. Pull AI model that you want to run with Ollama (example : ollama pull qwen3.5:4b)
  3. Run command ollama serve to make your local AI model available to this extension
  4. Add your favorite novel sites to the whitelist (click extension icon or this page)
  5. Configure your preferences on this page

Supported Websites:

  • webnovel.com — QiDian official English platform
  • fanmtl.com — Fan MTL novels
  • novelupdates.com — Novel tracker & database
  • wuxiaworld.com — Wuxia / xianxia licensed translations
  • royalroad.com — Original English web novels
  • scribblehub.com — Fiction community
  • lightnovelworld.org — JP/CN light novel translations
  • novelbin.me — Free novel aggregator
  • novelfire.net — Free web novels
  • wtr-lab.com — AI-translated novels
  • novelbuddy.com — Free novel reader
  • novelpub.com — Web novel platform

You can add any other site manually via the Whitelisted Sites tab.
